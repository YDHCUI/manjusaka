# manjusaka

https://github.com/ydhcui/manjusaka




## 牛屎花  
一款基于rust开发的远程主机管理系统


## 使用方法

1、添加监听器，上线地址改为外网IP

2、添加项目，回调地址改为外网IP，连接地址选刚才设置的监听器。

3、生成npc1 运行上线。

4、上线后连接npc1 加载npc2 等待回连。
